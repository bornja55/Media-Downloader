# 🚀 MEDIA CORE DL: The Ultimate Universal Downloader & Flipbook Extractor

แอปพลิเคชันเว็บแบบครบวงจรระดับพรีเมียม (Premium Web GUI) สำหรับดาวน์โหลดสื่อทุกประเภท ไม่ว่าจะเป็นวิดีโอ, เสียง หรือหนังสือ Flipbook ที่ถูกจำกัดสิทธิ์การเข้าถึง พร้อมด้วยระบบอัจฉริยะที่สามารถเข้าถึงเนื้อหาที่สงวนไว้เฉพาะสมาชิก (Member-Only) ได้อย่างเนียนกริบ!

---

## 🔥 ไฮไลท์ฟีเจอร์เด่น (Key Features)

### 🎥 1. ระบบดาวน์โหลดสื่อครอบจักรวาล (Universal Media Engine)
ดาวน์โหลดวิดีโอและเสียงด้วยความละเอียดสูงสุดเท่าที่มีให้โหลด (High-Quality Extraction) รองรับเว็บไซต์และแพลตฟอร์มหลายร้อยแห่งทั่วโลก ไม่ว่าจะเป็นแพลตฟอร์มสตรีมมิ่ง โซเชียลมีเดีย หรือเว็บวิดีโอเฉพาะทาง

### 📚 2. ระบบสกัดไฟล์ Flipbook ระดับพระกาฬ
ทลายข้อจำกัดของเว็บไซต์หนังสืออิเล็กทรอนิกส์! สกัดหน้าหนังสือที่ถูกล็อกเอาไว้จาก **FlipHTML5** และ **AnyFlip** นำมารวมเป็นไฟล์ PDF ที่มีความคมชัดสูง (High Resolution) แบบอัตโนมัติ 100% พร้อมตั้งชื่อไฟล์ตามชื่อหนังสือของจริงให้ทันที

### 🔑 3. ระบบยืนยันตัวตนอัจฉริยะ (VIP / Member-Only Bypass)
หมดยุคของการต้องมานั่งหาโปรแกรมอัดหน้าจอ! ด้วยเทคโนโลยี **Cookie Extraction** ระบบจะทำการยืมข้อมูลเซสชันอย่างปลอดภัยจากเบราว์เซอร์ในเครื่องของคุณ (รองรับ Chrome, Edge, Firefox, Brave, Safari, Opera) 
ทำให้คุณสามารถเข้าไปดึงไฟล์วิดีโอใน **กลุ่มปิด Facebook (Private Groups)**, **คลิปเฉพาะสมาชิก (Member-Only)** หรือ **คอร์สเรียนออนไลน์** ที่คุณมีสิทธิ์เข้าถึงอยู่แล้วได้โดยตรง เสมือนคุณกำลังนั่งกดโหลดเอง!

### ⚙️ 4. แดชบอร์ดตั้งค่าแบบรวมศูนย์ (Centralized Configuration)
ควบคุมทุกอย่างได้จากหน้าต่างตั้งค่า (Settings) รูปเฟืองมุมขวาบนของแอปพลิเคชัน:
- **Smart Output Directory:** คลิกปุ่มรูปแว่นขยาย (Browse Folder) เพื่อเลือกโฟลเดอร์ปลายทางในการบันทึกไฟล์ได้จากหน้าต่าง Windows Explorer โดยตรง
- **Dark / Light Mode:** สลับธีมหน้าจอระหว่างโหมดมืดสุดเท่ และโหมดสว่างสุดคลีนได้ด้วยคลิกเดียว (รองรับบนปุ่มลัดมุมขวาบน)
- **One-Click Maintenance:** ปุ่มสำหรับล้างประวัติ Log การทำงาน และปุ่มลบไฟล์ Cookie ชั่วคราว เพื่อความปลอดภัยและความเป็นส่วนตัวของคุณสูงสุด
- **Netscape Cookie Support:** สามารถใส่โค้ด Custom Cookie (รูปแบบ Netscape) ได้เองในหน้าหลัก สำหรับนักพัฒนาหรือผู้ใช้ระดับโปร

### ⚡ 5. ระบบประมวลผลคิวและ UI แบบเรียลไทม์ (Real-Time Batch Processing)
- ใส่ลิงก์กี่ร้อยลิงก์ก็ได้ ระบบจะจัดการดาวน์โหลดให้ทีละคิว (Batch Processing)
- หน้าจอ Terminal จำลองสุดล้ำ (Hacker-style GUI) ที่รายงานสถานะการทำงาน, เปอร์เซ็นต์การดาวน์โหลด, และความเร็วแบบวินาทีต่อวินาที
- รองรับการตั้งชื่อไฟล์แบบกำหนดเองด้วย Template (เช่น `media_export_%(title)s`)

## 🛠 สิ่งที่ต้องมีในเครื่อง (Prerequisites)

- **Python 3.12+**
- **Google Chrome** (จำเป็นต้องมีเพื่อใช้ร่วมกับระบบสกัดไฟล์ Flipbook PDF)
- *(ทางเลือก)* เว็บเบราว์เซอร์อื่นๆ หากต้องการใช้ระบบดูดคุกกี้เพื่อโหลดไฟล์ที่ติดสิทธิ์ส่วนตัว

## 📦 วิธีการติดตั้ง (Installation)

1. **โคลนซอร์สโค้ด (Clone Repository):**
   ```bash
   git clone <your-repo-url>
   cd "FB DL"
   ```

2. **ติดตั้งไลบรารีที่จำเป็น (Install Dependencies):**
   ```bash
   pip install -r requirements.txt
   ```

## 🎮 วิธีการเริ่มต้นใช้งาน (Quick Start)

1. **เปิดเซิร์ฟเวอร์หลังบ้าน:**
   รันสคริปต์หลักผ่านทาง Command Prompt หรือ PowerShell:
   ```bash
   python "FB DL.py"
   ```
2. **เข้าสู่หน้าต่างแดชบอร์ด:**
   เปิดเว็บเบราว์เซอร์ของคุณแล้วเข้าไปที่ URL:
   👉 `http://127.0.0.1:5000`

3. **ปรับแต่งการตั้งค่า (ก่อนโหลด):**
   - ไปที่ปุ่ม **ฟันเฟือง ⚙️** (มุมขวาบน)
   - กดปุ่มค้นหาโฟลเดอร์ 🔍 เพื่อกำหนดตำแหน่งที่คุณต้องการให้ไฟล์เซฟลงไป
   - สามารถเลือกเปลี่ยนเป็น **Light Mode ☀️** หรือ **Dark Mode 🌙** ได้ตามใจชอบ

4. **เริ่มกระบวนการดาวน์โหลด:**
   - เลือกโหมดจากแท็บด้านบน: **Video**, **Sound**, หรือ **PDF**
   - นำลิงก์เป้าหมายมาวางในกล่อง (วางได้หลายบรรทัด)
   - หากลิงก์นั้นเป็นเนื้อหาที่ต้องล็อกอิน ให้เลือก **"Browser for Cookie Extraction"** ให้ตรงกับเบราว์เซอร์ที่คุณล็อกอินทิ้งไว้
   - **(ใหม่!) การใช้ Custom Cookies:** หากคุณมีข้อมูล Cookie ดิบ (รูปแบบ Netscape) คุณสามารถนำข้อความมาวางในช่อง **"Custom Cookies"** ได้โดยตรง (ระบบจะให้ความสำคัญกับช่องนี้ก่อนการดึงจาก Browser เสมอ) 
     *(💡 แนะนำ: คุณสามารถใช้ส่วนขยาย (Extension) เช่น **"Get cookies.txt LOCALLY"** บน Chrome/Edge เพื่อก๊อปปี้ข้อมูลข้อความ Cookie มาวางได้เลย)*
   - กดปุ่ม **Download Queue** แล้วนั่งจิบกาแฟรอได้เลย! ☕

## ⚠️ ข้อควรระวังด้านความปลอดภัย (Security & Privacy)

- **Local Execution 100%:** โปรแกรมนี้รันบนเครื่องของคุณเท่านั้น ไม่มีการส่งข้อมูลรหัสผ่านหรือคุกกี้ของคุณไปยังเซิร์ฟเวอร์ภายนอกใดๆ ทั้งสิ้น ข้อมูลเซสชันคุกกี้ถูกอ่านแบบ "ชั่วคราว" เฉพาะตอนที่ต้องสกัดไฟล์เท่านั้น
- **Headless Chrome:** ระบบสกัด Flipbook จะแอบเปิด Google Chrome ทำงานอยู่เบื้องหลัง (ไม่รบกวนการทำงานหน้าจอ) กรุณาอย่าบังคับปิด Process ของ Chrome ใน Task Manager ขณะรันระบบ
