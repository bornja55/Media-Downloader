- **ระบบช่วยเหลือเมื่อตั้งค่าผิดพลาด**: หากคุณเผลอพิมพ์ชื่อโฟลเดอร์เซฟงานผิด หรือเลือกโฟลเดอร์ที่ไม่มีอยู่จริง โปรแกรมจะไม่ค้างหรือเด้งออกอีกต่อไป แต่จะช่วยแก้ปัญหาโดยเซฟงานลงในโฟลเดอร์สำรองให้แทนโดยอัตโนมัติ

## 🔥 ไฮไลท์ฟีเจอร์เด่น (Key Features)

### 🎥 1. ระบบดาวน์โหลดสื่อครอบจักรวาล (Universal Media Engine)
ดาวน์โหลดวิดีโอและเสียงด้วยความละเอียดสูงสุดเท่าที่มีให้โหลด (High-Quality Extraction) รองรับเว็บไซต์และแพลตฟอร์มหลายร้อยแห่งทั่วโลก ไม่ว่าจะเป็นแพลตฟอร์มสตรีมมิ่ง โซเชียลมีเดีย หรือเว็บวิดีโอเฉพาะทาง

### 📚 2. ระบบสกัดไฟล์ Flipbook ระดับพระกาฬ
ทลายข้อจำกัดของเว็บไซต์หนังสืออิเล็กทรอนิกส์! สกัดหน้าหนังสือที่ถูกล็อกเอาไว้จาก **FlipHTML5** และ **AnyFlip** นำมารวมเป็นไฟล์ PDF ที่มีความคมชัดสูง (High Resolution) แบบอัตโนมัติ 100% พร้อมตั้งชื่อไฟล์ตามชื่อหนังสือของจริงให้ทันที

### 🖼️ 3. ระบบดาวน์โหลดภาพและมังงะ (Image & Manga Downloader)
ดาวน์โหลดอัลบั้มภาพ หรือ โพสต์บนโซเชียลมีเดีย (เช่น Facebook) ได้ยกแผงในคลิกเดียว พร้อมรองรับการแปลงไฟล์เป็น 3 รูปแบบอัตโนมัติ:
- **Raw Images:** เซฟเป็นไฟล์ภาพเดี่ยวๆ เข้าโฟลเดอร์ให้เป็นระเบียบ
- **CBZ (Comic Book Archive):** มัดรวมภาพทั้งหมดเป็นไฟล์ `.cbz` สำหรับอ่านในแอพมังงะ
- **PDF Document:** แปลงรูปทั้งหมดเป็นไฟล์ PDF แผ่นเดียวจบด้วยความเร็วสูง

### 🔑 4. ระบบยืนยันตัวตนอัจฉริยะ (VIP / Member-Only Bypass)
หมดยุคของการต้องมานั่งหาโปรแกรมอัดหน้าจอ! ด้วยเทคโนโลยี **Cookie Extraction** ระบบจะทำการยืมข้อมูลเซสชันอย่างปลอดภัยจากเบราว์เซอร์ในเครื่องของคุณ (รองรับ Chrome, Edge, Firefox, Brave, Safari, Opera) 
ทำให้คุณสามารถเข้าไปดึงไฟล์วิดีโอใน **กลุ่มปิด Facebook (Private Groups)**, **คลิปเฉพาะสมาชิก (Member-Only)** หรือ **คอร์สเรียนออนไลน์** ที่คุณมีสิทธิ์เข้าถึงอยู่แล้วได้โดยตรง เสมือนคุณกำลังนั่งกดโหลดเอง!

### ⚙️ 5. แดชบอร์ดตั้งค่าและแถบควบคุม (Settings & Top Bar)
ควบคุมทุกอย่างได้จากแถบเมนูด้านบนขวาของแอปพลิเคชัน:
- **Dark / Light Mode:** สลับธีมหน้าจอระหว่างโหมดมืดสุดเท่ และโหมดสว่างสุดคลีนได้ทันทีด้วยปุ่มรูป ☀️/🌙 บนแถบด้านบน
- **หน้าต่างตั้งค่า (Settings ⚙️):**
  - **Smart Output Directory:** คลิกปุ่มรูปแว่นขยาย (Browse Folder) เพื่อเลือกโฟลเดอร์ปลายทางในการบันทึกไฟล์ได้จากหน้าต่าง Windows Explorer โดยตรง
- **One-Click Maintenance:** ปุ่มสำหรับล้างประวัติ Log การทำงาน และปุ่มลบไฟล์ Cookie ชั่วคราว เพื่อความปลอดภัยและความเป็นส่วนตัวของคุณสูงสุด
- **Netscape Cookie Support:** สามารถใส่โค้ด Custom Cookie (รูปแบบ Netscape) ได้เองในหน้าหลัก สำหรับนักพัฒนาหรือผู้ใช้ระดับโปร

### ⚡ 6. ระบบประมวลผลคิวและ UI แบบเรียลไทม์ (Real-Time Batch Processing)
- ใส่ลิงก์กี่ร้อยลิงก์ก็ได้ ระบบจะจัดการดาวน์โหลดให้ทีละคิว (Batch Processing)
- หน้าจอ Terminal จำลองสุดล้ำ (Hacker-style GUI) ที่รายงานสถานะการทำงาน, เปอร์เซ็นต์การดาวน์โหลด, และความเร็วแบบวินาทีต่อวินาที
- รองรับการตั้งชื่อไฟล์แบบกำหนดเองด้วย Template (เช่น `media_export_%(title)s`)

### 🤖 7. โซลูชันเจาะทะลุการบล็อกบันทึกหน้าจอ (Smart Screen Capture Bypass)
- **บันทึกภาพได้แม้เจอจอดำ:** หมดปัญหาแคปจอแล้วจอดำเวลาเปิดแอปพลิเคชันหรือเว็บ E-book บางแห่ง ระบบแคปเจอร์ของเราถูกออกแบบมาให้ทำงานร่วมกับหน้าต่างจำลอง (Virtual Machine) ทำให้สามารถดึงภาพเนื้อหาเหล่านั้นออกมาได้อย่างคมชัด 100%
- **จำลองการคลิกสุดเนียน:** ป้องกันการถูกแบนหรือจับได้ว่าเป็นบอท! ระบบจะสั่งการเมาส์และคีย์บอร์ดในระดับลึกที่แนบเนียนมาก เสมือนคุณกำลังนั่งคลิกพลิกหน้ากระดาษด้วยตัวเอง ทำให้ทะลวงผ่านระบบป้องกันของแอปต่างๆ ได้อย่างราบรื่น
- **ปลอดภัย ไม่เสี่ยงโดนแบน:** ไร้กังวลเรื่องการถูกระงับบัญชีผู้ใช้ เนื่องจากระบบของเราทำงานแยกส่วนออกมาอย่างสมบูรณ์ (เหมือนการตั้งกล้องถ่ายหน้าจอ) ทำให้แอปต้นทางไม่สามารถตรวจจับการดาวน์โหลดได้เลย

## 💿 รุ่นแบบพกพา (Standalone Executable)

โปรแกรมของเรามีให้เลือกใช้งานในรูปแบบไฟล์ `.exe` ที่สามารถคลิกใช้งานได้ทันทีโดยไม่ต้องติดตั้ง Python (One-Click) ซึ่งแบ่งออกเป็น 2 รุ่นหลักๆ ดังนี้:

### 1. `MediaCoreDL-Full.exe` (แนะนำ)
- **ขนาดไฟล์:** ~100MB+
- **จุดเด่น:** ใช้งานได้สมบูรณ์แบบที่สุด เพราะมีการฝังตัวโปรแกรม `ffmpeg` เอาไว้ภายในแล้ว 
- **ข้อดี:** เมื่อดาวน์โหลดคลิปจาก YouTube แบบความละเอียดสูง (1080p, 4K) ระบบจะสามารถมัดรวมไฟล์ภาพความละเอียดสูงและไฟล์เสียงเข้าด้วยกันได้ทันที ไร้ข้อจำกัดเรื่องคุณภาพไฟล์

### 2. `MediaCoreDL-Lite.exe` (รุ่นประหยัดพื้นที่)
- **ขนาดไฟล์:** ~35MB
- **จุดเด่น:** โหลดเร็ว ไฟล์เล็ก เหมาะกับการพกพา
- **ข้อจำกัด (ถ้าในเครื่องไม่มี ffmpeg):** หากคุณนำไปโหลดวิดีโอบน YouTube ที่มีความละเอียดตั้งแต่ 1080p ขึ้นไป ระบบของ YouTube จะบังคับแยกไฟล์ภาพและเสียงออกจากกัน เนื่องจากรุ่น Lite ไม่มี `ffmpeg` ในตัว โปรแกรมจะไม่สามารถรวมภาพและเสียงให้คุณได้ ระบบจึงจะทำการ Fallback ไปโหลดไฟล์คุณภาพสูงสุดที่มีทั้งภาพและเสียงในตัวเดียว (ซึ่งมักจะถูก YouTube จำกัดไว้ที่ **720p** สูงสุด)

> 💡 **สำหรับผู้ใช้รุ่น Lite ที่ต้องการโหลด 1080p+:**
> คุณสามารถติดตั้ง `ffmpeg` ด้วยตัวเองได้ง่ายๆ:
> 1. ไปที่เว็บไซต์ [FFmpeg Builds (by BtbN)](https://github.com/BtbN/FFmpeg-Builds/releases) 
> 2. ดาวน์โหลดไฟล์ `ffmpeg-master-latest-win64-gpl.zip`
> 3. แตกไฟล์ zip และเข้าไปในโฟลเดอร์ `bin`
> 4. ก๊อปปี้ไฟล์ `ffmpeg.exe` มาวางไว้ในโฟลเดอร์เดียวกับ `MediaCoreDL-Lite.exe` 
> เพียงเท่านี้ โปรแกรมก็จะมองเห็นและกลับมาโหลด 1080p+ ได้ตามปกติ!

## 📦 วิธีการติดตั้ง (Installation)

1. **โคลนซอร์สโค้ด (Clone Repository):**
   ```bash
   git clone <your-repo-url>
   cd "FB DL"
   ```

2. **ติดตั้งไลบรารีที่จำเป็น (Install Dependencies):**
   ```bash
   pip install -r requirements.txt
   ```

## 🎮 วิธีการเริ่มต้นใช้งาน (Quick Start)

1. **เปิดเซิร์ฟเวอร์หลังบ้าน:**
   รันสคริปต์หลักผ่านทาง Command Prompt หรือ PowerShell:
   ```bash
   python "FB DL.py"
   ```
2. **เข้าสู่หน้าต่างแดชบอร์ด:**
   เปิดเว็บเบราว์เซอร์ของคุณแล้วเข้าไปที่ URL:
   👉 `http://127.0.0.1:5000`

3. **ปรับแต่งการตั้งค่า (ก่อนโหลด):**
   - ไปที่ปุ่ม **ฟันเฟือง ⚙️** (มุมขวาบน)
   - กดปุ่มค้นหาโฟลเดอร์ 🔍 เพื่อกำหนดตำแหน่งที่คุณต้องการให้ไฟล์เซฟลงไป
   - สามารถเลือกเปลี่ยนเป็น **Light Mode ☀️** หรือ **Dark Mode 🌙** ได้ตามใจชอบ

4. **เริ่มกระบวนการดาวน์โหลด:**
   - เลือกโหมดจากแท็บด้านบน: **Video**, **Sound**, **Image**, หรือ **PDF**
   - นำลิงก์เป้าหมายมาวางในกล่อง (วางได้หลายบรรทัด)
   - **การใช้ Custom Cookies (จำเป็นสำหรับคลิป/โพสต์ Member-Only / Private Group!):** 
     หากต้องการดาวน์โหลดเนื้อหาที่ต้องล็อกอิน ให้คุณติดตั้งส่วนขยาย (Extension) ที่ชื่อว่า **[Get cookies.txt LOCALLY](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc)** บน Chrome/Edge 
     
     **🚨 สำคัญมาก:** คุณต้อง **เปิดหน้าแท็บวิดีโอ/ภาพ (YouTube, Facebook ฯลฯ) ที่สามารถดูได้ตามปกติค้างไว้ก่อน** แล้วจึงค่อยกดไอคอน Extension เพื่อ Copy ข้อมูลคุกกี้ดิบ (รูปแบบ Netscape)
     
     จากนั้นนำไปวางในช่อง **"Custom Cookies"** ในโปรแกรมได้โดยตรง (ระบบจะอิงจากคุกกี้ช่องนี้ โหลดผ่าน 100%!)

## 📁 ตำแหน่งการจัดเก็บไฟล์ (File Storage & Output)

หากคุณ **ไม่ได้ตั้งค่า** โฟลเดอร์ปลายทางในเมนู Settings โปรแกรมจะจัดเก็บไฟล์ตามค่าเริ่มต้นดังนี้:
- 📂 **โฟลเดอร์ปัจจุบันของโปรแกรม:** สำหรับเก็บไฟล์ **วิดีโอ (Video), เสียง (Sound), รูปภาพ (Image), และโหมด PDF**
- 📁 **`rpa_output/`** : สำหรับเก็บไฟล์ที่ได้จากการดูด Flipbook ผ่าน **ระบบ RPA (VM)** 
- 📁 **`dist/`** : (สำหรับนักพัฒนา) โฟลเดอร์สำหรับเก็บไฟล์โปรแกรม `.exe` ที่สั่งคอมไพล์

*(💡 **ข้อควรรู้:** แต่ถ้าหากคุณไปตั้งค่า Output Directory ใน Settings เป็นโฟลเดอร์อื่น (เช่น `C:\Downloads`) ไฟล์ดาวน์โหลด **ทุกประเภท (รวมถึง RPA)** จะถูกส่งไปรวมกันที่โฟลเดอร์นั้นทั้งหมดครับ)*

## ⚠️ ข้อควรระวังด้านความปลอดภัยและข้อจำกัด (Security & Privacy)

- **Local Execution 100%:** โปรแกรมนี้รันบนเครื่องของคุณเท่านั้น ไม่มีการส่งข้อมูลรหัสผ่านหรือคุกกี้ของคุณไปยังเซิร์ฟเวอร์ภายนอกใดๆ ทั้งสิ้น ข้อมูลเซสชันคุกกี้ถูกอ่านแบบ "ชั่วคราว" เฉพาะตอนที่ต้องสกัดไฟล์เท่านั้น
- **Headless Chrome:** ระบบสกัด Flipbook จะแอบเปิด Google Chrome ทำงานอยู่เบื้องหลัง (ไม่รบกวนการทำงานหน้าจอ) กรุณาอย่าบังคับปิด Process ของ Chrome ใน Task Manager ขณะรันระบบ
- **ข้อจำกัดของ Google Chrome (App-Bound Encryption):** เนื่องจากการอัปเดตความปลอดภัยของ Google Chrome การดึงคุกกี้อัตโนมัติอาจล้มเหลว (Error #7271) หากเกิดปัญหานี้ให้ใช้วิธี **ก๊อปปี้โค้ดมาวางในช่อง Custom Cookies** ด้วย Extension ตามที่ระบุไว้ในคู่มือด้านบน
